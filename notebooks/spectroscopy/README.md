# Spectroscopy Notebooks

This directory contains Jupyter notebooks related to spectroscopic techniques.

## Topics Covered

- Infrared (IR) spectroscopy
- Ultraviolet-Visible (UV-Vis) spectroscopy
- Nuclear Magnetic Resonance (NMR) spectroscopy
- Mass spectrometry
- Raman spectroscopy
- X-ray photoelectron spectroscopy (XPS)
- Beer-Lambert Law
- Spectral interpretation

## Learning Objectives

Students working through these notebooks will be able to:
- Interpret IR, UV-Vis, and NMR spectra
- Identify functional groups from spectroscopic data
- Apply Beer-Lambert Law to concentration determination
- Use spectroscopy for structural elucidation
- Understand the physical principles behind different spectroscopic techniques
